# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [v0.1.0] - 2026-02-26

### Added

- Express.js server with Helmet, Morgan, and CORS middleware
- REST API endpoints: GET `/`, GET `/api/about`, GET `/api/hello`
- React 19 frontend with TypeScript support
- Vite development server with HMR (Hot Module Replacement)
- Tailwind CSS 4.2.0 for styling
- shadcn/ui component library integration
- API Explorer page for interactive endpoint testing
- Header component with navigation and theme toggle
- Footer component with links and branding
- API client layer with `fetchEndpoint()` and `checkServerHealth()` functions
- Environment variable configuration with dotenv
- Responsive design with mobile-first approach
